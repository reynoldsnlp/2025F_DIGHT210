1. Comments - Adding Notes to Your Code

Comments are notes you write in your code that the computer ignores. They help you (and others) understand what your code does. It's like writing notes in the margins of a book.

There are two ways to write comments in Python:

  • Single-line comments: Start with a # symbol
  • Multi-line comments: Use triple quotes """ to write longer explanations

Example: Using Comments


# This is a single-line comment
print("Hello, World!")  # This comment is at the end of a line

"""This is a multi-line comment.
You can write as much as you want here.
The computer will ignore all of this text.
"""
print("Comments help make code readable!")

2. The print() Function - Displaying Information

The print() function is like a megaphone for your program - it displays information on the screen. You can print both numbers and text (called strings).

Important details:

  • Text must be surrounded by quotes: "Hello" or 'Hello'
  • Numbers don't need quotes: 42
  • Always include the parentheses: print()

Example: Printing Different Types of Information


# Printing text (strings)
print("Welcome to Python!")
print('Single quotes work too')

# Printing numbers
print(42)
print(3.14)

# What happens without parentheses?
print  # This shows the function itself, doesn't call it

3. Variable Assignment - Storing Information

Variables are like labeled boxes where you can store information. You can put a value in the box (assignment) and get the value out later.

To create a variable, use this pattern: variable_name = value

Rules for variable names:

  • Can contain letters, numbers, and underscores
  • Must start with a letter or underscore (not a number)
  • No spaces allowed (use underscores instead)
  • Case matters: name and Name are different

Example: Creating and Using Variables


# Storing different types of information
name = "Alice"
age = 25
height = 5.6
is_student = True

# Using variables
print(name)
print(age)
print("Height:", height)

# Variables can change
age = 26
print("New age:", age)

4. Math Operators - Doing Calculations

Python can do math just like a calculator. Here are the basic math operators:

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division (gives decimal result)
  • // Floor division (gives whole number result)
  • % Modulo (gives remainder after division)
  • ** Exponentiation (power)

Example: Math Operations


# Basic arithmetic
x = 10
y = 3

addition = x + y
subtraction = x - y
multiplication = x * y
division = x / y
floor_division = x // y
remainder = x % y
power = x ** y

print("Addition:", addition)
print("Subtraction:", subtraction)
print("Multiplication:", multiplication)
print("Division:", division)
print("Floor division:", floor_division)
print("Remainder:", remainder)
print("Power:", power)

5. Incrementing and Decrementing - Changing Variable Values

Incrementing means increasing a value, and decrementing means decreasing it. This is very common in programming.

There are two ways to do this:

  • Long way: x = x + 1 (add 1 to x and store back in x)
  • Short way: x += 1 (same thing, but shorter)

The short way works with all math operators: +=, -=, *=, /=

Example: Different Ways to Change Variables


# Starting value
score = 10
print("Starting score:", score)

# Long way to add 5
score = score + 5
print("After adding 5:", score)

# Short way to add 3
score += 3
print("After adding 3 more:", score)

# Short way to subtract 2
score -= 2
print("After subtracting 2:", score)

# Short way to multiply by 2
score *= 2
print("After doubling:", score)

# Short way to divide by 4
score /= 4
print("After dividing by 4:", score)

üéØ Bring It All Together: A Simple Calculator

Let's combine everything we've learned to create a simple calculator that shows our progress.

Example: Complete Calculator with Documentation


"""
Simple Calculator Program
This program demonstrates comments, variables,
math operations, and incrementing.
"""

# Starting values
num1 = 15
num2 = 4
result = 0  # This will store our answers

print("Calculator Demo")
print("First number:", num1)
print("Second number:", num2)

# Perform calculations
result = num1 + num2  # Addition
print("Addition result:", result)

result = num1 - num2  # Subtraction
print("Subtraction result:", result)

# Using increment operators
result = 0  # Reset to zero
result += num1  # Add first number
result += num2  # Add second number
print("Sum using increment:", result)

# Update result step by step
result *= 2  # Double it
print("After doubling:", result)

result -= 10  # Subtract 10
print("Final result:", result)
